# 数据标注系统 — 近期更新日志

---

## 使用前必读

> **1. 首次进入请强制刷新一次**
>
> 由于网站进行了较大改动，打开网页后请按 **Ctrl+F5** 强制刷新一次（清除浏览器缓存）。只需这一次，后面再进入不需要强制刷新。

> **2. 模板页标注完请稍等几秒再切页**
>
> 标注完一页模板后，**在页面上停留 5 秒左右**再切换到副本页。否则模板数据可能还没同步完成，副本页的信息会缺失。

---

## 一、键盘快捷翻页

| 按键 | 作用 |
|------|------|
| **→ 右箭头** | 跳转到下一单元 |
| **← 左箭头** | 跳转到上一单元 |

在输入框里打字时不会触发。

---

## 二、标注框拖拽和缩放

- **移动**：直接用鼠标拖拽任意标注框
- **缩放**：鼠标悬停在标注框上，四角出现白色方块，拖拽角落即可调整大小
- 题干框和作答框都支持拖拽和缩放
- 调整后自动保存

---

## 三、界面精简

| 删除内容 | 说明 |
|----------|------|
| 标记本页完成按钮 | 不再需要手动标记完成 |
| 学生得分输入框 | 主题目和子题均已删除 |
| 题目总分输入框 | 主题目和子题均已删除 |
| 备注输入框 | 主题目和子题均已删除 |
| 图片上传按钮 | 填空和问答题的标准答案、学生答案均不再支持上传图片 |

---

## 四、作答状态统一

所有题型（单选题、多选题、填空题、问答题）的作答状态选项统一为四个：
- **正确** / **错误** / **部分正确** / **空白**

---

## 六、副本页自动预填

标注完模板页后，切换到副本页时，以下内容会自动填入：

| 自动复制内容 | 默认值 |
|-------------|--------|
| 题干框位置 | 模板位置 |
| 作答框位置 | 模板位置 |
| 标准答案文本 | 副本已填入 |
| 学生答案文本 | 默认 = 标准答案 |
| 作答状态 | 默认 = **正确** |

学生只需**核查**，发现错题把状态改为「错误」，并将学生答案输入；或把状态改为「空白」即可，不需要每道题重新输入。

---

## 七、模板微调（偏移值）

右上角的 X/Y 偏移值现在**同时作用于题干框和作答框**，调整后两者整体移动，保持一致。

---

## 八、页码校正改为本地模式

页码校正不再通过云端同步，改为**本地文件管理**：

| 操作 | 说明 |
|------|------|
| 导入配置 JSON | 加载最新的页码校正配置文件 |
| 修改页码 | 改动仅保存在本地浏览器 |
| 导出修正后配置 | 下载更新后的配置文件 |
| 分享给他人 | 其他人导入你的文件即可获取最新页码 |

不再出现多人冲突覆盖的情况。

页码对齐工具栏已移至**左侧边栏**（「我的任务」下方），和右侧标注区分离，避免误触。
