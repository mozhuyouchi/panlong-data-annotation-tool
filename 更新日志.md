# 数据标注系统 — 近期更新日志

---

## 使用前必读

> **1. 首次进入请强制刷新一次**
>
> 由于网站进行了较大改动，打开网页后请按 **Ctrl+F5** 强制刷新一次（清除浏览器缓存）。只需这一次，后面再进入不需要强制刷新。

> **2. 模板页标注完请稍等几秒再切页**
>
> 标注完一页模板后，**在页面上停留 5 秒左右**再切换到副本页。否则模板数据可能还没同步完成，副本页的信息会缺失。

---

## 一、键盘快捷翻页

| 按键 | 作用 |
|------|------|
| **→ 右箭头** | 跳转到下一单元 |
| **← 左箭头** | 跳转到上一单元 |

在输入框里打字时不会触发。

---

## 二、标注框拖拽和缩放

- **移动**：直接用鼠标拖拽任意标注框
- **缩放**：鼠标悬停在标注框上，四角出现白色方块，拖拽角落即可调整大小
- 题干框和作答框都支持拖拽和缩放
- 调整后自动保存

---

## 三、界面精简

| 删除内容 | 说明 |
|----------|------|
| 标记本页完成按钮 | 不再需要手动标记完成 |
| 学生得分输入框 | 主题目和子题均已删除 |
| 题目总分输入框 | 主题目和子题均已删除 |
| 备注输入框 | 主题目和子题均已删除 |
| 图片上传按钮 | 填空和问答题的标准答案、学生答案均不再支持上传图片 |

---

## 四、作答状态统一

所有题型（单选题、多选题、填空题、问答题）的作答状态选项统一为四个：
- **正确** / **错误** / **部分正确** / **空白**

---

## 五、填空题多个作答区

填空题现在支持多个作答区：
- 点击题目下方的「**新增作答区**」按钮
- 在图片上框选新作答区
- 自动编号为「作答框 1」「作答框 2」...
- 每个作答区可独立调整

---

## 六、副本页自动预填

标注完模板页后，切换到副本页时，以下内容会自动填入：

| 自动复制内容 | 默认值 |
|-------------|--------|
| 题干框位置 | 模板位置 |
| 作答框位置 | 模板位置 |
| 标准答案文本 | 副本已填入 |
| 学生答案文本 | 默认 = 标准答案 |
| 作答状态 | 默认 = **正确** |

学生只需**核查**，发现错题把状态改为「错误」或「空白」即可，不需要每道题重新输入。

---

## 七、模板微调（偏移值）

右上角的 X/Y 偏移值现在**同时作用于题干框和作答框**，调整后两者整体移动，保持一致。

---

## 八、页码校正改为本地模式

页码校正不再通过云端同步，改为**本地文件管理**：

| 操作 | 说明 |
|------|------|
| 导入配置 JSON | 加载最新的页码校正配置文件 |
| 修改页码 | 改动仅保存在本地浏览器 |
| 导出修正后配置 | 下载更新后的配置文件 |
| 分享给他人 | 其他人导入你的文件即可获取最新页码 |

**优点**：不再出现多人冲突覆盖的情况。配置文件就是唯一权威版本。

页码对齐工具栏已移至**左侧边栏**（「我的任务」下方），和右侧标注区分离，避免误触。

---

## 九、标注数据同步说明

标注数据（题目框、答案、得分状态等）继续通过云端同步：

| 数据 | 同步方式 |
|------|---------|
| 页码校正 | 本地文件管理，不经过服务器 |
| 标注内容 | 修改后 5 秒内自动推送到服务器 |
| 他人数据 | 刷新页面后自动拉取 |

不同标注员的标注数据有独立编号，不会互相覆盖。

---

## 十、已知操作建议

1. **首次打开**：刷新一到两次确保数据加载完成
2. **数据不同步**：Ctrl+F5 强制刷新
3. **标注顺序**：标完一页模板 → 标该页所有副本 → 点右箭头进入下一页
4. **页码校正**：先导入最新配置文件，校正完导出给下一个同学
5. **检查数据**：换个浏览器打开网页，看数据是否已在云端
